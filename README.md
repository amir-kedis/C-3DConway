## Controls 

- `WASD+QE` or `mouse`: Move camera
- `P`: Pause simulation
- `R`: Randomize grid
- `C`: Change color shift type
- `SPACE`: Step simulation
- `ENTER`: Toggle Rotation
- `?`: Show Instructions
